# Job Application Tracker

A premium, modern web application for tracking job applications across multiple platforms.

## ✨ Features
- **Glassmorphism UI**: A sleek, dark-mode interface with frosted-glass effects.
- **Full-Stack**: Built with Python (Flask) and SQLite for persistent tracking.
- **Search & Filter**: Find applications easily by company or status.
- **Real-time Updates**: Add, update, and delete applications seamlessly.

## 🛠️ Tech Stack
- **Backend**: Flask, SQLAlchemy
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- **Database**: SQLite

## 🚀 Getting Started
1. Clone the repository.
2. Install dependencies: `pip install flask flask-sqlalchemy`
3. Run the app: `python app.py`
4. Visit `http://localhost:5000`
